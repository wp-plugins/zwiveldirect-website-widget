Zwivel widget WordpressPlugin. Lets you easily integrate your Zwivel doctor or GroupDirect widget.

== Description ==
Zwivel doctor widget Wordpress Plugin. Lets you easily integrate your Zwivel doctor widget. It is the same widget that you add by copy/pasting the code snippet from your Zwivel account settings. You control the look of the Zwivel widget in your Zwivel account settings. You can also copy/paste your GroupDirect link to get a GroupDirect widget

== Installation ==
1. Place the extracted `zwivel-widget` folder in the `/wp-content/plugins/` directory or use the repository to install it automatically from your site's WordPress dashboard
1. Activate the plugin through the *"Plugins"* menu in WordPress
1. Find the "Zwivel Widget" in **Appearance > Widgets** in your WordPress Dashboard, and drag it to the desired widget area
1. Copy/Paste the link to your ZwivelDirect profile in the widget's input box (you can find this page if you log into your Zwivel account and in the top right corner menu you select *"My ZwivelDirect Profile"*). Alternativelym you can Copy/Paste the link to your GroupDirect page (the link to which can be found in the header of your GroupDirect settings labeled as *"View Group Page"*)
1. Click *"Save"*
